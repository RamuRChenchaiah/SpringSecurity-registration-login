# SpringSecurity-registration-login
--------------------------------------------------

## Features:

* Register
* Registration confirmation (email)
* Login
* Forgot password (email)

--------------------------------------------------

* Spring (Version: 4.1.6) IOC, 
* Spring  Security (Version: 4.0.2), 
* Hibernate, 
* MysQL, 
* Guava (Version: 18.0)
